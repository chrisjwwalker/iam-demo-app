iam-demo
========

## How to run
Built using Typescript & NodeJS on node version 12.16.1

Install the dependencies
```shell script
yarn install
```

This runs the frontend app on port 8000
```shell script
yarn dev-app
```

This runs the api on port 8008
```shell script
yarn dev-api
```
